# Latex Template

This is a latex template for university.

## How to use

To use this template effectively please use these suggestions.

### Naming conventions

#### Labels

| Type       | Prefix      |
|------------|-------------|
| Section    | sec:<name>  |
| Subsection | ssec:<name> |
| Image      | img:<name>  |
| Table      | tab:<name>  |
| Figure     | fig:<name>  |
| Algorithm  | alg:<name>  |
| Chapter    | ch:<name>   |
| Equation   | eq:<name>   |
| Listing    | lst:<name>  |
| Appendix   | apx:<name>  |
